Walter- User Story 1

As a patient, I want to see available appointments so that I can schedule one.

As a patient, I want to receive notifications for my scheduled appointments so that I remember them.

As a patient, I want to choose what means of communication I am notified of my scheduled appointments.

As a patient, I want to send notifications of my appointments to others so they can choose to attend my appointments in case I need assistance.

As a patient, I want to be able to select what means of communications others are notified about my appointments so that it does not infringe on them.

As a patient, I want to be notified when someone has responded to my notification so that I can see if they can attend the appointment with me.

As a Bookee user, I want to be able to use biometrics to log in to my account, so that I don’t have to remember another login and password combination.

As a Bookee user, I want to see my scheduled appointments on my Bookee homepage.

Walter- User Story 2

As a Bookee user, I want to be able to save a list of providers to my home screen so I can easily select them.

As a Bookee user, I want to be able to see a provider’s schedule even if I already have an appointment bookee’d so I can see if another appointment has opened.

As a Bookee user, I want to be able to easily distinguish between taken and open appointment slots so I can see what is available.

As a Bookee user, I want to be able to see several weeks if not months of availability for appointments so I can select what works best for me.

As a Bookee user, I want to be notified if a provider has had a cancelation, so that I can see if that cancelled appointment fits my schedule better.

As a Bookee user, I want Bookee to ask if I want it to cancel an appointment if I change my appointment with the same provider so that I don’t forget too.

As a Bookee user, I want Bookie to confirm that I have cancelled an appointment by sending me a notification.

As a service provider, I want to be able to send notifications to customers of cancellations so I can try and fill that appointment.

As a service provider, I want to be able to send customers notification if I must cancel their appointment unfortunately.

As a service provider, I want to be able to provide customers with an option to schedule a new appointment if I must cancel their appointment.

Walter- User Story 3

As a Bookee user, I want login procedures to execute quickly in times that an appointment must be scheduled in an emergency.

As a Bookee user, I want to be able to add service providers to my homepage for quick access to their schedule.

As a Bookee user, I want to be able to search for new service providers, so I am aware of what’s options are available.

As a Bookee user, I want to be able to search for services providers by their specific names or a general description of the services they provide.

As a Bookee user, I want to be able to refine my providers searches to improve search results.

As a Bookee user, I want to be able to filter the provider search results by proximity and availability.

As a service provider, I want my service to be listed by name and under a generic service type on Bookee to improve customer traffic to my services.

As a service provider, I want my available appointments to be easily identifiable on the Bookee site.
