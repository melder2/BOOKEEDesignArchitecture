Scenarios

Appointment Availability Viewer
Home Features

Description:

As a patient, I want to see available appointments.

As a Bookee user, I want to see my scheduled appointments on my Bookee homepage.

As a Bookee user, I want to be able to easily distinguish between taken and open appointment slots so I can see what is available

As a Bookee user, I want to be able to see ahead several weeks, if not months, of availability for appointments so I can select what works best for me.

Constraints:

Some data provided by users or providers may be protected under the HIPAA privacy rule.

Comments:

Based on John, Walter, and Sherry’s scenarios
