John- User Story 1

As a doctor, I want to be able to open the application and be able to log in to my account for security purposes.

As a doctor, I want to be shown a calendar of appointments that show what appointments are taken and open.

As a doctor, I want to know what each appointment is regarding so that I can plan ahead.

As a doctor, I need to be able to tell who booked the appointment either one of my receptionists or one of my patients.

As a doctor, I want my receptionists and patients to only be able to book appointments in a certain time frame.

As a doctor, I want my receptionists to have more admin features when logging into the application.

As a patient, I want to schedule an appointment online.

As a patient, I want my scheduled appointment to be uploaded in the application so that the doctor’s office can see that I scheduled the appointment.

John- User Story 2

As a doctor and a receptionist, I want the application to be secure so that no sensitive information can be seen by people not authorized.

As a receptionist, I want my own log in for the application.

As a receptionist, I want to be able to mark out certain time frames during the day.

As a receptionist and a doctor, I want everyone to be able to book appointments as far out as a year.

As a surgery scheduler, I want my appointment to be marked differently than all the other appointments.

As a surgery scheduler, I want my appointments to only be able to be scheduled on certain days.

As a surgery scheduler, I want to have more of a detailed description of the appointment than the regular receptionists detailed description.

As a doctor and a receptionist, I want the application to send out a reminder of the appointment two days in advance.

John- User Story 3

As a patient, I want the application to be secure so that no sensitive information can be seen by people not authorized.

As a patient, I want to be able to book an appointment through the app.

As a patient, I want the application to be secure so that no sensitive information can be seen by people not authorized.

As a patient, I want to be able to cancel appointments through the app.

As a patient, I want to use my credentials from my google account.

As a patient, I want an appointment reminder two days before the appointment.

As a patient, I want a customer satisfaction survey after booking an appointment.

As a doctor and a receptionist, I would like to see the overall results of the survey.
